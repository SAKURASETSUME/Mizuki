```bash
#基本语法
df -h
```

## 查询指定目录的磁盘占用情况

```bash
#基本语法
du -h /目录
#查询指定目录的磁盘占用情况 默认为当前目录

-s #指定目录占用大小汇总
-h #带计量单位
-a #含文件 不加a只输出目录
--max-depth=1 #子目录深度
-c #列出明细的同时 增加汇总值
```

### 实例

```txt
查询/opt目录的磁盘占用情况 深度为1
```

```bash
du -h --max-depth=1 /opt
```
