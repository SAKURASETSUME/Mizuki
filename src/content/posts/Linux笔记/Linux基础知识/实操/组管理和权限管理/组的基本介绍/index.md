**Linux中每个用户必须属于一个组 不能独立于组外 在linux中每个文件有所有者 所在组 其它组的概念**

## 所有者

**一般为文件的创建者**

```bash
#查看文件的所有者
ls -ahl

#修改文件所有者
chown 用户名 文件名
```

## 组的创建

```bash
#创建组
groupadd 组名

#新建用户 并设置所在的组
useradd -g 组名 用户名
```

## 所在组

**当某个用户创建了一个文件后 这个文件的所在组就是该用户所在的组**

```bash
#查看所在组
ls -ahl

#修改所在组
chgrp 组名 文件名
```

## 修改所在组

```bash
#改变用户所在组
usermod -g 组名 用户名

#改变用户的初始登陆目录(用户需要有进入新目录的权限)
usermod -d 目录名 用户名
```
