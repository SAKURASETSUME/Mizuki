- ## pwd指令

```bash
#基本语法 作用：显示当前工作目录的绝对路径
pwd
```

- ##  ls指令

```bash
#基本语法
ls [选项] [目录或者文件]
#常用的参数有 -a（显示全部文件） -l（单列显示文件） -h（人类比较容易看的形式输出）
```

- ## cd指令

```bash
#基本语法
cd [参数]
#cd ~ 或者cd : 回到自己的家目录
#cd .. 回到上一级目录
```

- ## mkdir指令

```bash
#基本语法
mkdir [选项] 要创建的目录
#常用选项
-p 创建多级目录
```

- ## rmdir指令

```bash
#基本语法
rmdir [选项] 要删除的空目录
#如果要删除非空目录
rm -rf 目录
```

- ## touch指令

```bash
#基本语法 创建文件的
touch 文件名称
```

- ## cp指令

```bash
#基本语法 拷贝
cp [选项] source dest
#常用选项
-r 递归复制整个文件夹

#如果有需要覆盖的文件 批量同意 全部覆盖
\cp -r source dest
```

- ## rm指令

```bash
#基本语法 移除文件或目录
rm [选项] 文件或目录
#常用选项
-r #递归删除全部文件
-f #强制删除不提示
```

- ## mv 指令

```bash
#基本语法
mv oldNameFile newNameFile (重命名)
mv /temp/movefile /targetFolder (移动文件)
```

- ## cat指令

```bash
#基本语法
cat [选项] 要查看的文件
#常用选项
-n 显示行号
#为了浏览方便 一般加上管道命令 | more [进行交互]
```

- ## more指令

**more指令是基于vi编辑器的文本过滤器 它以全屏幕的方式按页显示文本文件的内容 more指令内置了若干快捷键 
空格键是翻页 
Enter键是下一行 
q键是退出 
CTRL+F向下滚动一屏 
Ctrl+B是返回上一屏 
=是输出当前的行号 
:f是输出文件名和当前行的行号**

```bash
#基本语法
more 文件名
```

- ## less指令

**用来分屏查看文件内容 功能与more类似 但是比more更强大 支持各种显示终端 less指令在显示文件内容时 并不是一次将整个文件加载后再显示 而是根据显示需要加载内容 对于显示大型文件有较高效率
快捷键：
空格 pagedown pageup都是切页
/字符串 向下寻找字符串 n是向下 N是向上 
?字符串 向上寻找字符串 
q离开**

```bash
#基本语法
less 文件名
```

- ## echo指令

```bash
#基本语法
echo [选项] [输出内容]

#输出环境变量
echo $PATH
#输出aaa
echo "aaa"
```

- ## head指令

**查看文件开头部分内容 默认显示10行**

```bash
#基本语法
head 文件
#显示5行
head -n 5 文件
```

- ## tail指令

**查看文件尾部内容 默认10行**

```bash
#基本语法
tail 文件
#显示5行
tail -n 5 文件
#实时追踪该文件的更新内容
tail -f 文件
```

- ## >和>>

**输出重定向 和 >>追加**

```bash
#基本语法
ls -l > 文件 （把列表的内容覆盖到文件中）
ls -al >> 文件 （把列表的内容追加到文件末尾中）
cat 文件1 > 文件2 (用文件1的内容覆盖文件2)
echo "内容" >> 文件

#将当前日历追加到/home/mycal
cal >> /home/mycal
```

- ## ln指令

**软链接 也称为符号链接 类似于Windows下的快捷方式 主要存放了链接其他文件的路径**

```bash
#基本语法
ln -s [原文件或目录] [软链接名]
```

- ## history指令

**查看已经执行过的历史命令 也可以执行历史指令**

```bash
#基本语法
history

#最近10个
history 10

#执行历史编号为5的指令
!5
```
