```bash
#获取帮助信息 空格可以翻到下一页 按q退出
man [命令或配置文件]

#例如
man ls  #查看ls的命令
#Linux下 隐藏文件是以.开头的 选项可以组合使用 比如ls -al或 ls -a -l

#获得shell内置命令的帮助信息
help 命令

#例如
help cd
```
