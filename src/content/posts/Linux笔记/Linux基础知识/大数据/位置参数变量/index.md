## 介绍

**当我们执行一个shell脚本时 如果希望获取到命令行的参数信息 就可以用到位置参数变量 比如:
./myshell.sh 100 200 这个就是一个执行shell的命令行 可以在myshell脚本中获取到参数信息**

## 基本语法

```bash
$n （功能描述：n为数字 $0代表命令本身 $1-$9代表第一个到第九个参数 十以上的参数需要用大括号包含 如${10}）
$* （功能描述：这个变量代表命令行中所有的参数 $*把所有的参数看作一个整体）
$@ （功能描述：这个变量也代表命令行中所有的参数 不过$@把每个参数分开对待）
$# （功能描述：这个变量代表命令行中所有参数的个数）
```

## 位置参数变量案例

```txt
编写一个shell脚本 position.sh 在脚本中获取到命令行的各个参数信息
```

```bash
#!/bin/bash
echo "0=$0 第一个参数=$1 第二个参数=$2"
echo "全部参数$*"
echo "$@"
echo "参数的个数=$#"                          
```

```bash
sh /root/shcode/myshell.sh 100 200
0=/root/shcode/myshell.sh 第一个参数=100 第二个参数=200
所有的参数100 200
100 200
参数的个数=2
```
