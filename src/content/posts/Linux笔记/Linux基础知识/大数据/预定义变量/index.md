## 基本介绍

**shell设计者事先已经定义好的变量 可以直接在shell脚本中使用**

## 基本语法

```bash
$$ （功能描述：当前进程的进程号）
$! （功能描述：后台运行的最后一个进程的进程号）
$? （功能描述：最后一次执行的命令的返回状态 如果这个变量的值为0 证明上一个命令正确执行 如果这个变量的值为非0 则证明上一个命令执行不正确）
```

## 案例

```txt
在一个shell脚本中简单实用预定义变量
preVar.sh
```

```bash
#!/bin/bash
echo "当前进程的进程号=$$"
#以后台的方式运行一个脚本 并获取他的进程号
/root/shcode/myshell.sh &
echo "后台运行的最后一个进程的进程号=$!"
echo "最后一次执行的命令的返回状态=$?"
```
